Το Monster RPG 2 είναι μια αποστολή φαντασίας που εκτείνεται σε ηπείρους και κόσμους και
σας επιτρέπει να πάρετε μια απλή χωρική και να την εξελίσσετε σε μια ηρωίδα που έχει τη
δύναμη να σώσει τον κόσμο της. Το επόμενο μέρος της κλασικής σειράς Monster
RPG, το Monster RPG 2, είναι ένα turn-based παιχνίδι ρόλων με
αρκετή ποικιλία, ανατροπές, μυστικά και σκηνικά που κάνουν ακόμα και τους
παραδοσιακούς παίκτες να επιστρέφουν για περισσότερο παιχνίδι. Εάν σας αρέσουν τα κλασικά 16-bit παιχνίδια
σε παλιές κονσόλες, θα λατρέψετε αυτό το παιχνίδι. Εάν δεν έχετε ακούσει για αυτά
τα παιχνίδια, ετοιμαστείτε για μια πραγματική απόλαυση!

Δυνατότητες παιχνιδιού ρόλων: 

- Δεκαεννιά τοποθεσίες για ανακάλυψη και εξερεύνηση.
- Εννιά εξερευνητές για προσθήκη στην ομάδα, ανάπτυξη και καθοδήγηση.
- Turn-based μάχες ενάντια σε πάνω από ενενήντα είδη τεράτων.
- Δεκαοκτώ ξόρκια για τους χρήστες μαγείας και τους κληρικούς (δεν περιλαμβάνονται
τα μαγικά που κάνουν τα τέρατα!)
- Αρκετά ξίφη, όπλα, μαγικά φίλτρα, κράνη, κονκάρδες, κλειδιά, σανίδες και
πανοπλίες για να διατηρήσετε κάθε εξερευνητή εφοδιασμένο στη ζωή.

Μπορεί να αναπαραχθεί στα Αγγλικά, τα Ολλανδικά, τα Γαλλικά, τα Γερμανικά, τα Ελληνικά, τα Ιταλικά, τα Πολωνικά, τα Πορτογαλικά Βραζιλίας, τα Πορτογαλικά Ευρώπης, τα Ρωσικά, τα Ισπανικά και τα κορεάτης.
